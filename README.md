# ONE in Marriage — Website

A pattern-aware marriage experience. Single self-contained `index.html` (all CSS/JS inline)
plus photos in `assets/images/`. No build step — open `index.html` in a browser, or host the
folder on any static host (GitHub Pages, Netlify, Vercel).

## Design
Matches the approved "Build a Marriage Filled with Safety, Connection, Light and Life" comp:
warm cream + sage palette, Lora (serif) + Outfit (sans) via Google Fonts.
